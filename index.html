3D NavMesh

A long term personal project still in development. The purpose is to create a dynamic Navigation Mesh for flying agents in three-dimensional space.
In ordrer to retrace the scene geometry with a low cost in empty areas, I'm using the octree data structure.
Then the A* algorithm implementation will find a quick path to the target location.

Implemented features :

•Generic octree data structure
•Geometry detection
•Finding neighbors cells algorithm
•Octree component ready for agent use

Features in progress :

•A* implementation
•Refresh navmesh only on geometry modification
