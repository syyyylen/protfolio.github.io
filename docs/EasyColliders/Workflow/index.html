Worflow

Here you can find documentation about the recommended workflow with Easy Colliders.

Easy Colliders is made to add quickly colliders to your level, placing Colliders Lines in a few clicks. For this, you will need to switch to the Placement mode.

To validate a Collider Line or to change from Placement to Selection mode, you can set up custom shortcuts inside your project settings. You can find them easily inside Project Settings if you search "collider".

It's heavily recommended to bind those shortcut and to use them to save a lot of time while placing your Colliders Lines.

Depending the size of the desired Collider, don't forget to set up a default height for the future colliders that you will place.

You can also fill the Colliders folder by any Folder name, existing or not, that will help you find them easily in the Level Hierarchy.
